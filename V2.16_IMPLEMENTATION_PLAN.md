## 🚀 MAJOR FEATURE: VOICE ECHO INTEGRATION

### What Voice Echo Does
Voice Echo is a **predictive pre-fetch system** that:
1. **Pre-loads related data** when you search (crosswalk, manufacturer, specs)
2. **Learns patterns** (after "HC9600", users usually ask "price" or "crosswalk")
3. **Deferred responses** for deep queries: "Give me a second while I look that up..."
4. **Caches predictions** so next queries are instant

### Current State
- ✅ Backend exists (`voice_echo.py`)
- ✅ Endpoints exist (`/api/voice-echo`, `/api/voice-echo-next`, etc.)
- ❌ **Not integrated** - Main app uses `/api/voice-search`

### Integration Plan

#### Step 1: Replace Voice Search Endpoint
**File:** `static/app.js`

Change from:
```javascript
var resp = await fetch(API_BASE + '/api/voice-search', { method: 'POST', body: formData });
```

To:
```javascript
var resp = await fetch(API_BASE + '/api/voice-echo', { 
    method: 'POST', 
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({
        query: transcript,
        session_id: sessionId,
        defer: true  // Enable deferred responses
    })
});
```

#### Step 2: Handle Deferred Responses
Add UI for "Give me a second while I look that up..." → Results appear after 10-15s

#### Step 3: Enable Predictive Cache
Pre-fetch crosswalk, manufacturer, and common follow-ups on every search

#### Step 4: Add "What Else" Support
Natural follow-up: User asks "What else?" → Returns next best prediction from cache

## 🧠 MODEL RECOMMENDATIONS

### Current Models
- **Router:** GPT-4.1-mini
- **Reasoning:** GPT-4.1

### Recommended Upgrades

| Use Case | Current | Recommended | Why |
|----------|---------|-------------|-----|
| Intent Classification | GPT-4.1-mini | **GPT-4.1-nano** | Faster, cheaper, sufficient for 3-category routing |
| Parameter Extraction | GPT-4.1-mini | **GPT-4.1-nano** | Structured extraction doesn't need large model |
| Conversational Response | GPT-4.1 | **o3-mini** | Better reasoning for nuanced recommendations |
| Chemical Analysis | GPT-4.1 | **Keep GPT-4.1** | Accuracy critical, keep proven model |

### Cost Impact
- GPT-4.1-nano: ~70% cheaper than GPT-4.1-mini
- o3-mini: Similar cost to GPT-4.1 but better reasoning
- **Overall:** Likely cost-neutral with quality improvement
