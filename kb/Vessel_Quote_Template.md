# Filtration/Separation Vessel Quote Template

Use this template when gathering requirements for vessel quotes. All fields should be collected before submitting to engineering.

**Contact:** the office | check in with the office

---

## 1. Customer Details

| Field             | Required | Description                  |
|-------------------|----------|------------------------------|
| Requestor Name    | Yes      | Person requesting quote      |
| Company           | Yes      | Company name                |
| Email             | Yes      | Contact email               |
| Phone             | Yes      | Contact phone               |
| Date              | Yes      | Request date                |

---

## 2. Project Reference

| Field              | Required | Description                     |
|--------------------|----------|---------------------------------|
| Project Ref/ID     | Yes      | Internal project reference      |
| End User           | Yes      | Final end user/customer         |
| Type of Quote      | Yes      | Budget, budgetary, firm         |
| Quote Deadline Date| No       | When quote is needed by         |

---

## 3. Process Conditions (Operating)

| Field                  | Required | Description                        |
|------------------------|----------|------------------------------------|
| Fluid                  | Yes      | What fluid is being filtered (attach SDS)|
| Flow Rate              | Yes      | GPM, BPD, or other units          |
| Operating Temperature  | Yes      | Normal operating temp             |
| Maximum Operating Temperature | Yes | Max temp the vessel will see      |
| Operating Pressure     | Yes      | Normal operating pressure (PSI)   |
| Type of Filtration/Separation | Yes | Particulate, coalescing, liquid-liquid, etc.|
| Vessel Orientation     | Yes      | Vertical or horizontal            |
| FTC Vessel Series      | No       | If known (SLV, HSSS, etc.)        |
| Vessel Quantity        | Yes      | Number of vessels needed          |

---

## 4. Process Conditions (Design)

| Field               | Required | Description                        |
|---------------------|----------|------------------------------------|
| Design Pressure     | Yes      | Usually higher than operating      |
| Design Temperature  | Yes      | Usually higher than operating      |
| Minimum Design Temperature| Yes | Lowest expected temp              |
| Corrosion Allowance | No       | Typically 1/16" or 1/8"            |
| Code Requirements   | Yes      | ASME Section VIII Div 1, etc.     |

## 5. Special Service Requirements (Yes/No)

| Field               | Default  | Notes                              |
|---------------------|----------|------------------------------------|

## 13. Special Notes

Free-form field for any additional requirements, constraints, or special instructions.

## Quote Checklist

Before submitting, verify:
1. All required fields completed
2. SDS attached for fluid
3. Customer contact info verified
4. Deadline date noted
5. Special service flags reviewed
